## 二、v2 核心理念：從「型錄網站」變「接單機器」

> 每一頁只做一件事：接住一種搜尋意圖，然後把人帶去詢價。

定位軸心（銨東真實強項，素材都在 session 記憶裡）：
**「小量多樣・難料（鈦/銅/不鏽鋼）・車銑複合」的台中精密加工夥伴**

## 三、改版內容

### 3.1 資訊架構重組（IA）
```
首頁(定位+3證據+CTA)
├─ 能力專頁×5 ← 新增，SEO 主戰場，每頁鎖一個字
│   ├─ 鈦金屬加工（純鈦車銑實績、難點怎麼解）
│   ├─ 黃銅/銅合金精密件（載盤/薄板案例）
│   ├─ 車銑複合加工（NT-2000LY 一次成型）
│   ├─ 不鏽鋼精密加工（薄件變形對策）
│   └─ 小量多樣/打樣試作（50-500件甜蜜點、報價快）
├─ 實績案例庫（v1 的 works 沿用＋持續擴充，按材料分類）
├─ 技術筆記 ← 新增，內容引擎（見3.2）
├─ 關於/品質/設備（v1 頁瘦身合併為信任頁）
└─ 聯絡詢價（表單＋報價三步驟＋24hr回覆承諾）
```

### 3.2 內容引擎：技術筆記（P3 解方）
把 session 裡累積的實戰知識**去識別化**寫成文章，每篇鎖一個長尾字：
- 「不鏽鋼薄件車削怎麼防變形」（白鐵薄件那套：全包夾/外徑不車）
- 「純鈦加工的切削條件與教訓」
- 「CNC 報價為什麼差這麼多——看懂加工報價的成本結構」
- 「小量試作找工廠的注意事項」
- 節奏：**每週 1 篇 × 8 週**，Claude 產草稿→傑哥 5 分鐘審→上線
- 這是 v2 與 v1 最大的差別：網站從「死的」變「活的」

### 3.3 技術 SEO（P1 解方）
- schema.org：LocalBusiness＋Service＋每案例頁 Product/CreativeWork
- 每頁 title/meta/H1 重寫（一頁一字）、hreflang 修正、sitemap 健檢
- 內部連結網：能力專頁 ↔ 案例 ↔ 技術筆記 三角互鏈
- 外鏈：名錄登錄（素材包已備）＋ Google 商家貼文連回專頁

### 3.4 轉換優化（P4 解方）
- 全站統一 CTA：「上傳圖面，24 小時內回覆報價」
- 聯絡頁加「報價三步驟」（傳圖→報價→打樣）＋常見問題
- 名片/報價單 QR（已做）指到 /contact 帶 UTM
- Formspree 每月 50 封夠用，破量再升級

### 3.5 三語策略收斂（P5 解方）
- **中文全力做**（客源實際在台灣轉單圈）
- 英文保留但只維護：首頁＋能力頁＋案例（RFQ 導向，配合日後 Etsy/海外）
- 日文降級為「有就好」：只留首頁＋聯絡（省 11 頁維護），除非日系客源出現

### 3.6 量測升級（P6 解方）
- traffic-report.mjs 擴充：per-page 真人流量、來源字、表單提交數 → **月報自動化**
- 每月一次「官網月報」進 memory，數字驅動下一步
