# 銨東官網專案啟動 — 2026-04-30

## 起因

傑哥提出新任務：做一個可以公開的公司官網。需求列點：
- 增加曝光度，SEO 關鍵字搜尋要做
- 公司門面要漂亮
- 詢價表單必須有，統一寄到公司信箱
- 素材：照片有但是隨手拍可能醜，傑哥另外想辦法
- 想先在內網確認後再上架，網路安全有疑慮，不希望網站跑在現有電腦

## 規劃決策過程

### 參考
傑哥提供 `https://www.te2017.com/index.php`（東研精密）做為視覺/結構參考。WebFetch 分析後認為東研那站的結構非常標準（首頁/關於/設備/加工項目/實績/品質/聯絡），可以直接抄資訊架構。

### 技術選型
**靜態網站 + 雲端部署**，原因：
- 安全：無資料庫、無後端、攻擊面極小
- 不用主機：傑哥電腦/文書機都不用 24/7 開
- 免費：Cloudflare Pages 免費額度遠超需求
- SEO：靜態頁速度快、Google 友善

選 **Astro** 框架（不是純 HTML 也不是 Next.js）：
- 學習成本低、產出純 HTML/CSS/JS
- Component 可重用（Layout 抽出）
- 內建 SEO 友善
- Cloudflare Pages 一鍵部署

### 部署架構決策
1. 開發階段：本機 `npm run dev` 跑 http://localhost:4321
2. 確認 OK push 到 GitHub
3. Cloudflare Pages 自動 build + 部署
4. DNS 切到 adcnc.com.tw（傑哥去買的網域）
5. 詢價表單接 Formspree（免費 SaaS，繞過自架後端）

### 信箱問題（重要）
傑哥從創業就用 `joineve0913@gmail.com`，老客戶都熟，不能換。
解法：**Cloudflare Email Routing 免費** — 註冊 `info@adcnc.com.tw` 自動轉寄到 Gmail，傑哥還是用熟悉的 Gmail 收信，但對外秀專業信箱。

## 網域決策

DNS 查詢：
- `ad.com.tw` 已被註冊（解析 210.65.129.218）
- `adcnc.com.tw` 可註冊 ✓
- `ad-cnc.com.tw` / `andongcnc.com.tw` 也可用但不推薦

選 **adcnc.com.tw**：
- 含 cnc 對 SEO 加分（搜「AD CNC」「銨東 CNC」直擊）
- 比純 ad.com.tw 更有業界辨識度（ad 太通用）
- 維持 AD 品牌縮寫

註冊商選 HiNet（傑哥習慣）：
- 一年 NT$850
- **只勾「網域註冊」**，所有加值（主機、信箱、DNS）都不勾
- DNS 後續改指 Cloudflare 接管

## SEO 布局

**關鍵字**（傑哥列的）：
- 地區：CNC 加工 台中 / 台中太平 CNC
- 工法：車銑複合加工、車床加工、銑床加工、三軸銑床加工、四軸銑床加工、五軸加工
- 材料：鑄鋁加工、鑄鐵加工、不鏽鋼加工、鋼材加工、鋁材加工、板材加工

**布局**：
- 每頁獨立 title / description / keywords meta
- JSON-LD LocalBusiness（Google 抓地理位置 + 公司資料）
- 關鍵字自然出現在 H1/H2/段落內，不堆砌
- og:url / og:image 設好分享預覽
- `lang="zh-TW"` + `<meta name="google" content="notranslate">`（避免 Chrome 自動翻譯）

## RWD 三斷點

- `>900px`：桌機，nav 全展開
- `≤900px`：平板/手機，漢堡選單
- `≤600px`：小手機，浮動按鈕變圓形圖示、hero 字級縮小、按鈕直排
