## 2. 各頁內容大綱

### `/about` 關於銨東
- **公司沿革**（時間軸）：創立年份 → 重要里程碑 → 引進機台 → 數位化（V3 內網系統）→ 機聯網
- **創辦故事**：傑哥本人 20+ 年加工經驗，創立銨東股份有限公司的初衷
- **公司資料卡**：統編 83181576、地址、電話、傳真、信箱
- **Google Maps 嵌入**：太平區成功路 29 巷 22 號
- **照片**：公司外觀 1 張、廠房全景 1 張

### `/services` 加工項目（SEO 主力）
- 6 個項目各一個專區，每區包含：
  - 大標 + 簡介 200 字
  - 適用零件範例（清單 + 照片）
  - 適用材質
  - 精度範圍 / 尺寸範圍
  - 內嵌「立即詢價」按鈕
- **SEO 關鍵字布局**：每區的 H2 / 段落自然帶入「車銑複合加工」「五軸加工」「三軸銑床加工」⋯⋯

### `/equipment` 設備介紹
- 7 台機台一台一張卡（廠牌 / 型號 / 軸數 / 加工範圍 / 控制器 / 照片）
- 量測設備（如有）
- **照片需求**：每台機台 1 張正面照 + 1 張運轉中／加工中照
- 列出：永進機械 NSV102am、台中精機 VII-20、NSV106 五軸、NT-2000LY、NP16、VLL#1（其他傑哥補上）

### `/works` 實績案例
- 卡片網格 4 欄
- 每張卡：加工件照片 + 產業類別（汽車 / 機械 / 自動化 / 治具）+ 材質 + 一句話描述
- 客戶名稱可不公開（很多 NDA），但「服務客戶包含 XXX 產業」可寫
- **照片需求**：12-20 個加工件特寫照（背景乾淨）

### `/quality` 品質政策
- 品質方針（一句話 slogan）
- 品檢流程圖（首件 → 過程 → 出貨）
- 檢驗設備清單（三次元、投影機、卡尺、輪廓儀⋯⋯ 傑哥列）
- 如有 ISO 認證，把證書照片放這
- 機聯網即時生產追蹤（V3 系統介紹，秀截圖 1-2 張）

### `/contact` 聯絡詢價
- **詢價表單**（送 Formspree → 轉發到 info@adcnc.com.tw → 傑哥 Gmail）
  - 必填：聯絡人、電話、E-mail、詢價內容
  - 選填：公司、預估數量、交期需求、上傳圖檔
- 公司資料區：地址、電話、傳真、E-mail、營業時間
- LINE 官方帳號 QR Code（大張）
- Google Maps 嵌入
- 「立即撥打」「LINE 諮詢」大按鈕

## 3. 拍照 Shot List

### 必拍（沒拍網站撐不起來）

**A. 公司外觀**
- 公司門面正面 1 張（招牌清楚）
- 公司門面 45 度斜角 1 張
- 公司大門 + 招牌特寫
- 廠房外觀全景

**B. 廠房內部**
- 廠房全景 2-3 張（不同角度，要乾淨整齊，事先整理）
- 加工區走道
- 半成品 / 成品擺放區

**C. 機台（每台 2 角度）**
- 7 台機台，每台拍 2 張：
  1. 機台正面全機（含廠牌型號）
  2. 機台運轉中或主軸特寫
- 控制器面板特寫（顯示加工程式畫面）

**D. 加工件實績照（12-20 張）**
- 不同材質代表（鑄鋁、鑄鐵、不鏽鋼、鋁合金、鋼）
- 不同加工類型代表（車削件、銑削件、複雜件）
- 拍攝建議：白色或灰色乾淨背景、自然光 + 補光、單件特寫 + 群件擺放

**E. 品檢 / 檢驗**
- 三次元量測進行中
- 卡尺 / 投影機檢查特寫
- QC 室全景

**F. 工作日常**
- 操作員操作機台中（戴護目鏡 / 手套，安全形象）
- 圖面對照加工
- 出貨包裝

### 加分（有時間就拍）
- 公司大合照
- 機聯網看板螢幕
- 深夜加班空無一人但機台運轉的場景（很有戲劇感，但要小心曝光）

### 拍攝技術建議
- **設備**：手機就夠（iPhone 12 以上 / 中高階 Android）。能借單眼更好
- **光線**：白天大光圈 + 開閃光燈補陰影；機台油污反光多，盡量側光不要直打
- **構圖**：機台拍正面要對齊水平、垂直；零件照背景單純（一張白紙 / 灰紙板）
- **格式**：原檔 JPEG 高品質，**至少 3000px 寬**（之後縮小不失真）
- **整理**：拍完照按資料夾分類（公司外觀 / 廠房 / 機台 / 加工件 / 品檢 / 日常）

### 還沒拍前先用什麼？
- **Stock 圖暫代**：Unsplash 有大量工業 CNC 照片可以免費商用
- **方向**：先用 stock 把版面填起來給你看效果，等你拍好真照再換上
